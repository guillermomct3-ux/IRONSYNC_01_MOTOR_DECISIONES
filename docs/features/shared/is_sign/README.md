# IS Sign

## Estado
DISENADO

## Que es
Capacidad transversal de firma digital.
Permite capturar firma de residente/supervisor via WhatsApp o web.

## Para que sirve
Validar que operador y residente confirman hechos registrados.
Requerido para cobro ante Mota-Engil.

## Modulos que lo usan
- logbook: firma residente en PDF Reporte Diario
- finanzas: firma conciliacion mensual
- gemelo_digital: firma entrega-recepcion maquina

## Evidencia
pendiente de documentar
